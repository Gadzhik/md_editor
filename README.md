# Markdown Editor V4 (AI Powered)

Современный, быстрый и функциональный редактор Markdown, созданный на базе Electron. Позволяет работать с локальными файлами, проектами и использовать возможности искусственного интеллекта для написания и редактирования текстов.

## Основные возможности

-   **Полноценный Markdown**: поддержка GFM, таблиц, списков задач.
-   **Математика и Диаграммы**: рендеринг формул через **KaTeX** и диаграмм через **Mermaid**.
-   **Управление проектами**: древовидный проводник (Explorer) для работы с папками.
-   **Система вкладок**: работайте с несколькими файлами одновременно.
-   **Интеграция с AI (V4)**:
    -   **AI Chat**: боковая панель для обсуждения текста.
    -   **Inline Assistant**: выделяйте текст и запрашивайте улучшения (Rewrite, Explain, Summarize) прямо в редакторе.
    -   Поддержка локальных LLM через **Ollama** или **LM Studio** (OpenAI-совместимый API).
-   **Git Auto-versioning**: автоматические коммиты при сохранении (если папка является Git-репозиторием).
-   **Экспорт**: сохранение заметок в форматы **PDF** и **HTML**.
-   **Zen Mode**: режим фокусировки (F11) для работы без отвлекающих факторов.
-   **Кастомизация UI**: возможность скрывать боковую панель, редактор или окно предпросмотра для настройки идеального рабочего пространства.
-   **Гибкие настройки**: быстрая смена провайдера и автоматическое переключение адреса сервера (порт 11434 для Ollama, 1234 для LM Studio).

## Установка и запуск

### Предварительные требования
1.  Установленный [Node.js](https://nodejs.org/) (рекомендуется LTS версия).
2.  (Опционально) Установленный [Ollama](https://ollama.com/) или [LM Studio](https://lmstudio.ai/) для работы локального AI.

### Шаги установки
1.  Склонируйте или скачайте репозиторий.
2.  Откройте терминал в папке проекта.
3.  Установите зависимости:
    ```bash
    npm install
    ```
4.  Запустите приложение:
    ```bash
    npm start
    ```

## Горячие клавиши

-   `Ctrl + O`: Открыть файл.
-   `Ctrl + Shift + O`: Открыть папку проекта.
-   `Ctrl + S`: Сохранить текущий файл.
-   `Ctrl + F`: Поиск по тексту.
-   `Ctrl + B`: Сделать выделенный текст жирным.
-   `Ctrl + I`: Сделать выделенный текст курсивом.
-   `F11`: Переключить Zen Mode (режим фокусировки).
-   `Ctrl + E`: Показать/скрыть Редактор.
-   `Ctrl + Shift + B`: Показать/скрыть Боковую панель (Sidebar).
-   `Ctrl + Shift + P`: Показать/скрыть Окно предпросмотра (Preview).
-   `Esc`: Закрыть модальные окна (настройки AI).

## Настройка AI

Редактор поддерживает два основных способа подключения локального AI:

### Вариант А: Ollama (по умолчанию)
1.  Убедитесь, что **Ollama** запущена.
2.  В приложении откройте настройки AI (⚙️).
3.  Выберите провайдера **Ollama**. Адрес сервера автоматически установится на `http://localhost:11434`.

### Вариант Б: LM Studio
1.  Запустите **LM Studio** и включите Local Server.
2.  В приложении откройте настройки AI (⚙️).
3.  Выберите провайдера **LM Studio / Compatible**. Адрес сервера автоматически изменится на `http://localhost:1234`.

### Общие шаги
1.  После выбора провайдера нажмите **Test Connection & Fetch Models**.
2.  Выберите нужную модель из появившегося списка.
3.  Нажмите **Save & Close**.
